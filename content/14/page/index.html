MegaMan Clone

Clone de MEGAMAN (CAPCOM), desenvolvido em CONSTRUCT 2 no ano de 2014, como parte dos exercícios de LEVEL DESIGN para o curso de DESENVOLVIMENTO DE JOGOS DIGITAIS. CONSTRUCT 2 é uma ferramenta da SCIRRA.COM, que foca no desenvolvimento rápido de jogos, criada para aqueles que adorariam desenvolver games mas não tem tempo ou paciência para aprender programação.
