100 MTS Run

“100 MTS Run” é um jogo que nasceu PHP Java Script em um projeto de Pós Graduação, cresceu Facebook e atingiu a maioridade no Construct 2. Pressione as setas do teclado alternativamente para correr o mais rápido possível, superando a si mesmo a cada rodada. Use o itens disponíveis com o dinheiro que ganhou na corrida anterior e seja feliz …
